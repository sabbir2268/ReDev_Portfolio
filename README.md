<h2>Portfolio Website</h2>

A personal portfolio website showcasing projects, skills, and professional experience. Built with React, Vite, and Tailwind CSS, this responsive web application is mobile-first and performance-optimized.

<h2>Features</h2>
Fully responsive design for desktop, tablet, and mobile.
Project showcase section with detailed descriptions.
Contact form with email integration.
Navbar with smooth scrolling to different sections.
Modern UI with Tailwind CSS styling.


<h2>Tech Stack</h2>
Frontend: React, Vite, Tailwind CSS, HTML, CSS
Routing: React Router
Version Control: Git & GitHub

<h2>Installation & Setup</h2>
Clone the repository
git clone https://github.com/sabbir2268/ReDev_Portfolio.git

Navigate to the project directory
cd your-repo-name

Install dependencies
npm install

Start the development server
npm run dev

Open the browser at the displayed local URL (usually http://localhost:5173).

Build for Production
npm run build

The production-ready files will be generated in the dist/ folder.
